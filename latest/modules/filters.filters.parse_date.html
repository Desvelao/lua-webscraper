Module filters.filters.parse_date

Functions

apply (v, fmt)
Parses a date string into ISO format (YYYY-MM-DD). Supported format specifiers: %Y (year), %m (month), %d (day).

Parameters:

  • v The input date string to be parsed.
  • fmt The format of the input date string (default is "%d/%m/%Y").

Returns:

    The date in ISO format (YYYY-MM-DD) or the original string if parsing fails.
